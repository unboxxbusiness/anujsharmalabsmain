# **App Name**: Anuj Sharma

## Core Features:

- About Me: Showcase Anuj Sharma's background and expertise in AI Automation and Digital Marketing.
- Services & Solutions: Highlight Anuj Sharma's professional offerings.
- Portfolio & Case Studies: Demonstrate Anuj Sharma's achievements through selected case studies.
- Contact Form: Allow potential clients to contact Anuj Sharma directly.

## Style Guidelines:

- Primary color: Modern Blue (#29ABE2) for a professional and trustworthy feel.
- Background color: Light Grey (#F0F0F0) for a clean and modern look.
- Accent color: Dark Grey (#555555) for text and important UI elements.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, for a clean and modern feel.
- Use professional, minimalist icons to represent services and skills.
- Clean, structured layout with clear sections to enhance readability and user experience.
- Subtle transition animations to create a smooth and engaging user experience.
